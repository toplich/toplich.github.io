MySQL Basics Cheat Sheet

This article contains essential commands for working with MySQL: creating users, databases, tables, managing access, and viewing information.

1. Connect to MySQL

1

mysql -u root -p

Or with host specified:

1

mysql -u username -p -h server_ip

2. Database Management

Create a database:

1

CREATE DATABASE mydb;

Show all databases:

1

SHOW DATABASES;

Select a database:

1

USE mydb;

Drop a database:

1

DROP DATABASE mydb;

3. Table Management

Create a table:

1
2
3
4
5

CREATE TABLE users (
  id INT AUTO_INCREMENT PRIMARY KEY,
  name VARCHAR(50),
  email VARCHAR(100)
);

Show tables:

1

SHOW TABLES;

Describe table structure:

1

DESCRIBE users;

Drop a table:

1

DROP TABLE users;

4. Basic Queries

Insert data:

1

INSERT INTO users (name, email) VALUES ('John Doe', 'john@example.com');

Select data:

1

SELECT * FROM users;

Update data:

1

UPDATE users SET email = 'new@example.com' WHERE id = 1;

Delete data:

1

DELETE FROM users WHERE id = 1;

5. Users and Permissions

Create a user:

1

CREATE USER 'username'@'%' IDENTIFIED BY 'password';

Grant all privileges on a database:

1

GRANT ALL PRIVILEGES ON mydb.* TO 'username'@'%';

Grant read-only access to a specific table:

1

GRANT SELECT ON mydb.mytable TO 'username'@'%';

Check user privileges:

1

SHOW GRANTS FOR 'username'@'%';

Revoke all privileges:

1

REVOKE ALL PRIVILEGES ON mydb.* FROM 'username'@'%';

Drop a user:

1

DROP USER 'username'@'%';

6. Administration

Show running processes:

1

SHOW PROCESSLIST;

Reload privileges:

1

FLUSH PRIVILEGES;

Exit MySQL:

1

EXIT;
